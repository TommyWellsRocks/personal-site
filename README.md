## Get started

1. Clone the repo: 
``git clone https://github.com/TommyWellsRocks/tommywells.me.git``
2. Install dependencies:
``pnpm install`` or npm install / yarn install
3. Run the development server:
``pnpm dev`` or npm run dev / yarn dev
